# vanity-bot-discord

Reward users for having vanity in bio (also supports emojis that contain the vanity url)
Requires presence intent and is against the Discord Terms of Service (so you cant get it verified but unverified sure)

# Installation instructions
- Run NPM install or use a managed host to install required packages
- Import database.sql into a mysql database
- Fill the required details in config.js
- Run the script
